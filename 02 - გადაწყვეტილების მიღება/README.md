## მათემატიკური ოპერატორები დარტში
მათემატიკურ ოპერატორებს შენ უკვე იცნობ. პროგრამირებაში მათ იგივე დანიშნულებით ვიყენებთ, თუმცა გვაქვს მციდედი განსხვავებებიც.

## კომენტარები
შემდგომ მაგალითებში დარტის კოდს თან ახლავს ეგრედ წოდებული კომენტარები. კომენტარი იწყება ორი დახრილი ხაზით (//) და ის არ არის კოდის ნაწილი. ანუ დარტი კომენტარს არ შეასრულებს როგორც ინსტრუქციას. კომენტარები მხოლოდ ადამიანებისთვის არის. 


### შეკრება

 - ორი მთელი(int) რიცხვის შეკრება გვაძლევს მთელ რიცხვს
 - მთელი(ინტ) და ათწილადი(double) რიცხვების შეკრება გვაძლევს ათწილადს
 - ორი ათწილადის შეკრება გვაძლევს ათწილადს ან მთელ რიცხვს

```dart
    void main() {
        print(4 + 3); // დაბეჭდავს 7_ს
        print(4.5 + 3); // დაბეჭდავს 7.5
        print(3.5 + 2.4); // დაბეჭდავს 5.9_ს
        print(3.2 + 2.8); // დაბეჭდავს 6_ს
    }
```


### სხვაობა
 - ორი მთელი(int) რიცხვის სხვაობა გვაძლევს მთელ რიცხვს
 - მთელი(ინტ) და ათწილადი(double) რიცხვების სხვაობა გვაძლევს ათწილადს
 - ორი ათწილადის სხვაობა გვაძლევს ათწილადს ან მთელ რიცხვს


```dart
    void main() {
        print(4 - 3); // დაბეჭდავს 1_ს
        print(4.5 - 3); // დაბეჭდავს 1.5
        print(3.4 - 2.5); // დაბეჭდავს 0.8999999999999999_ს. 
        print(3.9 - 2.9); // დაბეჭდავს 1_ს
    }
```
!!! იქიდან გამომდინარე, თუ როგორ ვინახავთ ათწილად რიცხვებს კომპიუტერის მახსოვრობაში ათწილადებზე ოპერაციამ შეიძლება დააბრუნოს ოდნავ არასწორი რიცხვი. თუმცა სწორ პასუხთან განსხვავება მინიმალურია და შეგვიძლია არ მივაქციოთ ყურადღება უმეტეს პროგრამებში. თუ კი ეს სიზუსტე მნიშვნელოვანია ჩვენი პროგრამისათვის(მაგ. ფინანსური აპლიკაცია) დარტი ისევე როგორც სხვა პროგრამირების ენები გვაძლევს სპეციალურ ხელსაწყოებს ზუსტი ოპერაციებისათვის.

### გამრავლება
- ორი მთელი(int) რიცხვის ნამრავლი გვაძლევს მთელ რიცხვს
- მთელი(ინტ) და ათწილადი(double) რიცხვების ნამრავლი გვაძლევს ათწილადს ან მთელ რიცხვს
- ორი ათწილადის ნამრავლი გვაძლევს ათწილადს ან მთელ რიცხვს

```dart
    void main() {
        print(4 * 3); // დაბეჭდავს 12_ს
        print(2.5 * 3); // დაბეჭდავს 7.5
        print(2.5 * 4); // დაბეჭდავს 10
        print(2.5 * 3.2); // დაბეჭდავს 8_ს. 
        print(2.5 * 3.3); // დაბეჭდავს 8.25_ს
    }
```

### გაყოფა
დარტში გაყოფის შესასრულებლად ორი ოპერატორი გვაქვს / (დახრილი ხაზი) და ~/ 
 - / - გაყოფა მოსალოდნელ შედეგს გვაძლევს ისევე როგორც მათემატიკაში
 - ~/ - ეგრედწოდებული მთელი გაყოფა. ორი რიცხვის გაყოფისას გვიბრუნებს მხოლოდ მთელ ნაწილს.

```dart
    void main() {
        print(4 / 2); // დაბეჭავს 2_ს
        print(4.2 / 2); // დაბეჭდავს 2.1_ს
        print(8.8 / 2.2); // დაბეჭდავს 4_ს
        print(4 ~/ 2); // დაბეჭდავს 2_ს
        print(4.2 ~/ 2); // დაბეჭდავს 2_ს
        print(4.2 ~/ 3); // დაბეჭდავს 1_ს
        
    }
```

### ნაშთი
ზოგადად პროგრამირებაში და რათქმაუნდა დარტშიც ასევე გვაქვს ნაშთის ოპერატორი - % (პროცენტის ნიშანი). მისი გამოყენებისას გვიბრუნდება ერთი რიცხვის მეორეზე გაყოფისას მიღებული ნაშთი

```dart
    void main() {
        print(4 % 3); // დაბეჭდავს 1_ს
        print(4 % 2); // დაბეჭდავს 0_ს
        print(12 % 9); // დაბეჭდავს 3_ს
    }
```


## boolean მონაცემის ტიპი
განსხვავებით აქამდე ნასწავლი მონაცემის ტიპებისაგან boolean მონაცემის ტიპს შეუძლია შეინახოს მხოლოდ ორი მნიშვნელობა. true(ჭეშმარიტი) ან false(მცდარი).
როგორც წესი boolean მონაცემის ტიპს პროგრამირებაში ვიყენებთ რაიმე ინფორმაციის ჭეშმარიტობის, არსებობის ან შედარების დასადგენად. მაგალითად: არის თუ არა მზიანი ამინდი, არის თუ არა ერთი რიცხვი მეტი მეორეზე, შეიცავს თუ არა ფაილი ინფორმაციას და სხვა.

```dart
    void main() {
        bool isTeenager = false;
        var isEven = true;
        print(isTeenager); // დაბეჭდავს false_ს
        print(isEven); // დაბეჭდავს true_ს
    }
```

## შედარების ოპერატორები
შედარების ოპერატორები ადარებს ორ ინფორმაციას ერთმანეთს და ყოველთვის აბრუნებს boolean ტიპის მნიშვნელობას (true ან false).

 - ტოლობის ოპერატორი == (ორი უდრის სიმბოლო) ადარებს მარცხნივ მდგომ ინფორმაციას მის მარჯვნივ მდგომ ინფორმაციასთან თუ მნიშვნელობები ტოლია აბრუნებს true_ს
 წინააღმდეგ შემთხვევაში false_ს

 ```dart
    void main() {
        int a = 3;
        int b = 4;
        print(a == b); // დაბეჭდავს false_ს
        b = 3;
        print(a == b); // დაბეჭდავს true_s
    }
```
 - მეტობის ოპერატორი > . ადარებს მარცხნივ მდგომ ინფორმაციას მის მარჯვნივ მდგომ ინფორმაციასთან, თუ მარცხნივ მდგომი მნიშვნელობა მეტია აბრუნებს true_ს ყველა სხვა 
 შემთხვევაში false_ს
 
 ```dart
    void main() {
        
        print(3 > 4); // დაბეჭდავს false_ს
        print(4 > 3); // დაბეჭდავს true_ს
        print(4 > 4); // დაბეჭდავს false_ს
    }
```

 - ნაკლებობის ოპერატორი < . ადარებს მარცხნივ მდგომ ინფორმაციას მის მარჯვნივ მდგომ ინფორმაციასთან, თუ მარცხნივ მდგომი მნიშვნელობა ნაკლებია აბრუნებს true_ს ყველა სხვა შემთხვევაში false_ს 

```dart
    void main() {
        
        print(3 < 4); // დაბეჭდავს true_ს
        print(4 < 3); // დაბეჭდავს false_ს
        print(4 < 4); // დაბეჭდავს false_ს
    }
```

 - მეტია ან ტოლია ოპერატორი >= . ადარებს მარცხნივ მდგომ ინფორმაციას მის მარჯვნივ მდგომ ინფორმაციასთან, თუ მარცხნივ მდგომი მნიშვნელობა მეტია ან ტოლი აბრუნებს true_ს თუ ნაკლებია false_ს

 ```dart
    void main() {
        
        print(3 >= 4); // დაბეჭდავს false_ს
        print(4 >= 3); // დაბეჭდავს true_ს
        print(4 >= 4); // დაბეჭდავს true_ს
    }
```

 - ნაკლებია ან ტოლია ოპერატორი <= . ადარებს მარცხნივ მდგომ ინფორმაციას მის მარჯვნივ მდგომ ინფორმაციასთან, თუ მარცხნივ მდგომი მნიშვნელობა ნაკლებია ან ტოლი აბრუნებს true_ს თუ მეტია false_ს

 ```dart
    void main() {
        
        print(3 <= 4); // დაბეჭდავს true_ს
        print(4 <= 3); // დაბეჭდავს false_ს
        print(4 <= 4); // დაბეჭდავს false_ს
    }
```

## გადაწყვეტილების მიღება პროგრამირებაში

შედარების ოპერატორებისა და boolean მონაცემის ტიპის დახმარებით პროგრამის წერისას შეგვიძლია მივიღოთ გადაწყვეტილება. რაიმე პირობის მიხედვით შევასრულოთ ერთი ან მეორე ინსტუქცია(ები).

```dart
    void main() {
        int number = 10;

        if (number > 0) {
            print('დადებითი რიცხვი');
        } else {
            print('უარყოფითი რიცხვი');
        }
    }
```

ზემოთ მოცემული კოდის გაშვების შედეგად დაიბეჭდება 'დადებითი რიცხვი' რადგან if პირობაში გამოსახულების (number > 0) შედეგი არის true ამიტომ შესრულდა if _ის ტანში არსებული ინსტუქცია(შეიძლება გვქონდეს ერთზე მეტი ინსტრუქციაც).

```dart
    void main() {
        int number = -10;

        if (number > 0) {
            print('დადებითი რიცხვი');
        } else {
            print('უარყოფითი რიცხვი');
        }
    }
```

ზემოთ მოცემული კოდის გაშვების შედეგად დაიბეჭდება 'უარყოფითი რიცხვი' რადგან if პირობაში გამოსახულების (number > 0) შედეგი არის false ამიტომ შესრულდა else _ის ტანში არსებული ინსტუქცია(შეიძლება გვქონდეს ერთზე მეტი ინსტრუქციაც).


## ლოგიკური 'და'(&&) და 'ან'(||) ოპერატორები
ხშირად პროგრამის წერისას, გვჭირდება ერთზე მეტი პირობის შემოწმება, რათა გადავწყვიტოთ რომელი ინსტუქცია(ები) შევასრულოთ. ამისთვის დარტში გვაქვს კიდევ ორი ოპერატორი
ესენია 'და'(&&) და 'ან'(||)

 - && ოპერატორი ამოწმებს მის მარცხინ და მარჯვნივ არსებულ გამოსახულებებს(ამ გამოსახულებებმა აუცილებლად boolean მონაცემის ტიპის მნიშნველობა უნდა დააბრუნონ) და აბრუნებს true_ს მხოლოდ იმ შემთხვევაში თუკი მარჯვენა და მარცხენა გამოსახულებები არის true. ყველა სხვა შემთხვევაში აბრუნებს false


```dart
    void main() {
        int a = 9;
        int b = 15;

        if (a > 0 && b > 0) {
            print('ორივე რიცხვი დადებითია');
        } else {
            print('ერთერთი რიცხვი მაინც არ არის დადებითი');
        }
    }
```

ზემოთ მოცემულ კოდი გაშვებისას დაბეჭდავს: 'ორივე რიცხვი დადებითია'

- || ოპერატორი ამოწმებს მის მარცხინ და მარჯვნივ არსებულ გამოსახულებებს(ამ გამოსახულებებმა აუცილებლად boolean მონაცემის ტიპის მნიშნველობა უნდა დააბრუნონ) და აბრუნებს true_ს იმ შემთხვევაში თუკი მარჯვენა ან მარცხენა გამოსახულება არის true. || ოპერატორი დააბრუნებს false მნიშნველობას მხოლოდ მაშინ თუკი ორივე პირობა არის false.

```dart
    void main() {
        int a = 9;
        int b = -15;

        if (a > 0 || b > 0) {
            print('ერთერთი რიცხვი მაინც დადებითია');
        } else {
            print('ორივე რიცხვი უარყოფითია');
        }
    }
```

ზემოთ მოცემულ კოდი გაშვებისას დაბეჭდავს: 'ერთერთი რიცხვი მაინც დადებითია'


## სავარჯიშოები
